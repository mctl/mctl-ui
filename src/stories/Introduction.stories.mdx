# mctl-ui

一个开源react组件库，支持多种基础组件

### 安装
~~~javascript
yarn add mctl-ui
//或
npm install mctl-ui
~~~

### 使用
~~~javascript
// 加载样式
import 'mctl-ui/dist/index.css'
// 引入组件
import { Button } from 'mctl-ui'
~~~
