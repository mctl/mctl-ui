
## mctl-ui component library

一个React组件库，使用React Hooks + TypeScript构建

### 1、文档地址
http://81.69.13.224/mctl-ui

### 2、安装
~~~javascript
yarn add mctl-ui
~~~

### 2、使用
~~~javascript
// 加载样式
import 'mctl-ui/dist/index.css'
// 引入组件
import { Button } from 'vikingship'
~~~

### 本地开发命令
~~~javascript
//启动本地文档服务
yarn storybook

//启动单元测试
yarn test

//本地打包
yarn build

//发布到npm
npm publish
~~~
